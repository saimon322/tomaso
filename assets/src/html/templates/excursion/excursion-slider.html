Экскурсия для
потенциальных партнеров
В экскурсию входит прогулка по ферме, а также рассказ о процессах выращивания животных без антибиотиков, проверок здоровья и качества мяса.
Экскурсия
для франчайзи
Прогулка по ферме, детальный разбор Системы Тимошовка, возможностей и перспектив создания бизнеса без антибиотиков. Возможна отдельная консультация по вопросам бизнес-процессов и цикла производства.
Экскурсия
для журналистов
Обзор фермы, возможность отснять фото- и видеоматериалы, а также время для интервью.
Экскурсия
для всех желающих
Экскурсия по отдельным частям фермы, рассказ о системе работы. Экскурсии не проводятся на регулярной основе.
